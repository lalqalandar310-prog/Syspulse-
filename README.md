# Syspulse
🛡️SysPulse – The Silent Guardian of Your Windows PC

SysPulse is a lightweight security monitor that watches your system
silently and sends you instant Telegram alerts when something
unusual happens.

You run it. It disappears into the background. And from that moment,
you know exactly what is happening on your PC.

Why people buy SysPulse

They want to know when a new program starts, with its full file path.

They want to be alerted the moment someone plugs in a USB drive.

They want to be warned if Windows Defender suddenly gets disabled.

They want to monitor CPU, RAM and Disk usage without opening Task
Manager every five minutes.

They want all of this delivered to their phone, not to another
dashboard they will forget to check.

What makes SysPulse different

No console window. No tray icon. No noise. Just alerts on Telegram.

It uses less than 30 MB of RAM while running.

Smart alerts that never spam you. Every alert is sent only once
until the situation returns to normal.

A built-in whitelist so you can ignore trusted processes.

A daily report that sums up everything that happened.

One-click setup. No installers. No configuration wizards.

How it works

You download the ZIP file, extract it, and edit a single file
with your license key and Telegram bot details.

You double-click SysPulse.exe. The program waits 30 seconds to
learn your system, then starts watching.

From that moment, you receive Telegram messages like this one:

New Process Detected
Name: notepad.exe
Path: C:\Windows\System32\notepad.exe
RAM: 12.5 MB
Time: 14:32:18

You can stop it anytime with a single command, and it never
touches your personal files or network traffic beyond the
Telegram API.

What you get

A single executable file that runs on Windows 10 and Windows 11.

A configuration file where you set your own thresholds for CPU,
memory and disk usage.

The ability to enable or disable USB monitoring, Startup
monitoring, Defender checks and daily reports.

A lifetime license that belongs to you, not a subscription.

System Requirements

Windows 10 or Windows 11 (64-bit). Less than 30 MB of RAM while
running. An internet connection only when alerts are sent.
